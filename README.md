# 🌦️ Weather App — CI/CD Pipeline with GitHub Actions

> A production-style DevOps project implementing an automated CI/CD pipeline for a React weather application using GitHub Actions.

---

## 🚀 Project Overview

This project demonstrates end-to-end CI/CD automation for a React-based weather application. The focus is on the **DevOps pipeline implementation** — automated testing, building, and deployment triggered on every push and pull request.

The weather app itself is a React PWA that tracks real-time atmospheric conditions for any city or country, with 5-day forecasts at 3-hour intervals.

---

## ⚙️ CI/CD Pipeline — What I Built

| Stage | Tool | Description |
|---|---|---|
| Source Control | GitHub | Feature branch workflow with PR-based merges |
| CI Trigger | GitHub Actions | Workflows triggered on push & pull_request |
| Build | Node.js + npm | Installs dependencies and builds the React app |
| Test | Jest | Automated unit tests on every commit |
| Deploy | Vercel / GitHub Pages | Auto-deploy on merge to main |

### Pipeline Highlights
- ✅ Automated build and test on every pull request
- ✅ Zero-downtime deployments on merge to `main`
- ✅ Workflow badges for live pipeline status visibility
- ✅ Secrets managed via GitHub Encrypted Secrets
